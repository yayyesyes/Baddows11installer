Baddows 11

Baddows 11 is a system designed to make fun of Windows 11, but better.

Also the desktop looks better on Arch Linux

 

Baddows 11 install guide:

Make sure you have git installed, yay, and an Arch Linux system installed to your drive. Also sudo too.

Git clone the install script's repository.

git clone https://github.com/yayyesyes/Baddows11installer.git

Next, cd to the folder:

cd Baddows11installer

Run the install script (run ls to find the script name, then type ./ and type in the script name then run.)

./baddows.sh

Once you ran the script, follow the instructions (The install script will ask for your password for sudo)

The installer will ask for your username ON THE SYSTEM.

Wait and your done. Cinnamon will automatically load once done
